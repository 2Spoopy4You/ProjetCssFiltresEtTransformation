Transform : matrix

La fonction CSS matrix permet d'incliner, changer l'échelle, faire une rotation, décaler un élément HTML. Cette fonction regroupe les élements scale(), skew() et translate() et en une seule fonction.

La fonction matrix() se présente de cette manière : transform : matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY())
